
# Torus

A graphical demonstration written in Common Lisp. Not very sophisticated, but
that doesn't stop it from looking kinda cool.

## Requirements

- sbcl (probably)
- quicklisp (recommended)
- lispbuilder-sdl
- cl-opengl

## To run

> git clone into directory and cd

> sbcl --load "torus.asd"

> (ql:quickload :torus-demo)

> (torus-demo::main)
